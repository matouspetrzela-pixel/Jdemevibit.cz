Složka pro uložení screenshotů (např. vizuál sekce Projekty z localhostu).

Jak uložit vizuál sekce Projekty z localu:
1. Spusťte lokální server: npm run dev
2. Otevřete v prohlížeči: http://localhost:3000/use-cases
3. Pořiďte snímek obrazovky:
   - Windows: Win + Shift + S (výřez) nebo Print Screen
   - Nebo F12 > DevTools > Command Palette (Ctrl+Shift+P) > "Capture full size screenshot"
4. Uložte soubor sem, např.: projekty-sekce-local.png
